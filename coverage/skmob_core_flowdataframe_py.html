24

32  

33 destination : str, optional 

34 the name of the column in `data` containing the destination location. The default is `constants.DESTINATION`. 

35  

36 flow : str, optional 

37 the name of the column in `data` containing the flow between two locations. The default is `constants.FLOW`. 

38  

39 datetime : str, optional
