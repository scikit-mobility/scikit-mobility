96 Parameters 

97 ---------- 

98 traj : TrajDataFrame 

99 the trajectories of the individual. 

100

103

105 ------- 

106 float

360def _true_entropy(sequence): 

361 n = len(sequence) 

362 

363 # these are the first and last elements 

364 sum_lambda = 1. + 2. 

365 

366 def in_seq(a, b): 

367 for i in range(len(a) - len(b) + 1): 

368 valid = True 

369 for j, v in enumerate(b): 

370 if a[i + j] != v: 

371 valid = False 

372 break 

373 if valid: return True 

374 return False 

375 

376 for i in range(1, n - 1): 

377 j = i + 1 

378 while j < n and in_seq(sequence[:i], sequence[i:j]): 

379 j += 1 

380 if j == n: j += 1 # EOF character 

381 sum_lambda += j - i 

382 

383 return 1. / sum_lambda * n * np.log2(n)

1157  

1158 as_ranks : boolean, optional 

1159 if True, return a list where element :math:`i` indicates the average visitation frequency of the :math:`i`-th most frequent location. The default is False.

1162 if True, show a progress bar. The default is True. 

1163

1165 the name of the column(s) indicating the location. The default is [constants.LATITUDE, constants.LONGITUDE].

1168 ------- 

1169 pandas DataFrame or list

1188 >>> lf_df = location_frequency(tdf, normalize=True).reset_index() # frequencies ad probabilities 

1189 >>> print(lf_df.head())

1191 0 0 39.762146 -104.982480 0.101953 

1192 1 0 39.891077 -105.068532 0.065269

1202 visits_per_location

1211 if n_users == 1: # if there is only one user in the TrajDataFrame

1213 return df.reset_index()

1216 if show_progress: 

1217 df = pd.DataFrame(traj.groupby(constants.UID) 

1218 .progress_apply(lambda x: _location_frequency_individual(x, normalize=normalize, location_columns=location_columns))) 

1219 else:
